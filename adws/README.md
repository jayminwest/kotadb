# AI Developer Workflow Scripts

This directory houses workflow scaffolding for the automated developer system. Each subdirectory focuses on a specific stage of the workflow lifecycle.

- `triggers/`: Functions and handlers for external webhooks that start workflows.
- `environments/`: Container definitions and runtime manifests per workflow category.
- `prompts/`: Prompt templates and metaprompts consumed by agent runners.
- `reviews/`: Scripts for generating, validating, and publishing code reviews or PR updates.
